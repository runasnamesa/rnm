# 📖 Stack Oficial do Runas na Mesa

> "Uma boa taverna não é construída pela quantidade de ferramentas, mas pela harmonia entre elas."

## Objetivo
Definir as ferramentas oficiais do projeto e evitar complexidade desnecessária.

## Stack Oficial
- Substack — Conteúdo
- VS Code — Desenvolvimento
- Astro — Site
- GitHub — Versionamento
- Vercel — Hospedagem
- n8n — Automações
- Cloudflare — DNS e Segurança
- Antigravity — Assistente de código
- ChatGPT — Arquitetura, UX, documentação

## Regra de Ouro
Antes de adicionar uma ferramenta:
1. Qual problema ela resolve?
2. Já existe algo que faça isso?
3. Vale a pena aprender mais uma tecnologia?
