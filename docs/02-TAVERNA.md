# 📜 Livro da Taverna

## Missão
Criar um espaço acolhedor para jogadores e mestres de RPG.

## Atmosfera
Uma taverna quente após uma longa travessia por uma montanha nevada.

## Inspirações
- Celeste
- Journey
- Spiritfarer
- Bibliotecas antigas
- Taverns medievais

## O que queremos
- Acolhimento
- Leitura confortável
- Pouca poluição visual
- Comunidade

## O que evitamos
- Clickbait
- Popups invasivos
- Publicidade agressiva
